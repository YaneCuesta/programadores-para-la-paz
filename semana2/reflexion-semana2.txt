Reflexión – Semana 2

Aprender a utilizar la terminal es una habilidad fundamental para cualquier desarrollador, ya que permite trabajar de manera más rápida y organizada. Mediante comandos es posible navegar entre carpetas, crear archivos y administrar proyectos sin depender de una interfaz gráfica. Además, el uso de la terminal facilita la automatización de tareas repetitivas y mejora la productividad. También es una herramienta indispensable para trabajar con Git y gestionar el historial de los proyectos. Conocer estos comandos permite solucionar problemas de forma más eficiente y comprender mejor el funcionamiento del sistema operativo. Considero que dominar la terminal es un paso importante para desarrollar proyectos de software de manera profesional. Esta práctica me ayudó a familiarizarme con comandos básicos que seguiré utilizando en futuras actividades y proyectos.
